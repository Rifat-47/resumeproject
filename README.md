## Run Locally

Clone the project

```bash
  git clone https://github.com/Rifat-47/resumeproject.git
```

Go to the project directory

```bash
  cd resumeproject
```

Install dependencies

```bash
  python
  django
```

Start the server

```bash
  python manage.py runserver
```
